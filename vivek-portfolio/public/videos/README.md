## Tips for smooth playback

- Export at 1080p, H.264, and keep hover-preview clips under ~15MB each —
  they autoplay muted on hover/tap, so smaller files load instantly.
- If you'd rather use YouTube instead of local files, see the "YouTube"
  section in the root `README.md`.
